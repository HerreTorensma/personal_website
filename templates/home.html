My name is not saying it, I study computer science and pay way too much rent for a tiny room.

Currently my main creative interest is programming game engines. My current project, zinc95, is a fantasy PC inspired by programs like PICO-8 and TIC-80, but with MS-DOS to Windows 95 era capabilities. I also want to get back into 3D graphics programming, but this project needs to be finished first. Besides this I'm interested in lower level programming and eventually want to create my own programming language and operating system, and play around some more with electronics.
